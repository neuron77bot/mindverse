# MindMap - Estado Mental Personal

Aplicación web para visualizar y gestionar el estado mental de una persona a través de mapas mentales interactivos, permitiendo explorar diferentes dimensiones de la vida en pasado, presente y futuro.

## Descripción del Proyecto

MindMap es una herramienta de autoconocimiento que permite a los usuarios crear representaciones visuales de su estado mental organizadas por:

- **Dimensión temporal**: Pasado, Presente y Futuro
- **Categorías de vida**: Salud, Trabajo, Amor, Finanzas, Familia, Desarrollo Personal, Ocio, etc.

Los usuarios pueden crear nodos, establecer conexiones entre pensamientos, y visualizar patrones en su bienestar emocional y mental.
